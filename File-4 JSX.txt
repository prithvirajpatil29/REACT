JSX - Javascript XML
    1. JSX - Syntax extension for Javascript
    2. We  can write markup inside the Javascript file.
    3. We will have a function which will return method.
        inside the return 
            -> we will write our markup/HTML code
    4. We will use the curely braces to define variables inside the markup 
    5. We can use variables inside the JSX -> using the {}
    6. Important -> return statement
                        -> never return more than 1 element
                 -> always should return 1 element
    7. We can use Objects and map there value inside the markup.
    8. You can call functions and pass parameters to the function inside the markup.
